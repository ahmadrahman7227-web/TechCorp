# 🚀 TechCorp

Modern IT Company Profile Website with AI Assistant  
Built using React, Vite, Tailwind CSS, and Framer Motion.

---

## ✨ Features

- 🤖 AI Assistant Chatbot (Interactive)
- 🎬 Animated Hero Section
- 🌐 Modern Company Profile Layout
- 🧑‍💻 Team Showcase
- 📰 Blog Section
- 📱 Fully Responsive Design
- ⚡ Smooth Animations (Framer Motion)

---

## 🛠️ Tech Stack

- React.js
- Vite
- Tailwind CSS
- Framer Motion
